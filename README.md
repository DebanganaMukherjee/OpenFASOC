## Running the Flow:
To change to temp-sen-gen directory and run the flow, open Terminal and type the folowing commands:

```
$ cd /OpenFASOC/openfasoc
$ cd /generators/temp-sense-gen/ 
$ sudo make sky130hd_temp
```
